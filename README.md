📌 Description

This is a simple C++ console-based Number Guessing Game where the program generates a random number and the user has to guess it. After each guess, the program tells whether the guess is too high or too low until the correct number is found. It is a beginner-friendly project that helps improve logic building and understanding of basic programming concepts like loops, conditions, and random number generation.

✨ Features

Random number generation each time the game runs
User input-based guessing system
Hints like “Too High” or “Too Low”
Unlimited attempts until correct guess
Simple and interactive console gameplay
🛠️ Technologies Used

C++
iostream library
cstdlib (for random number generation)
ctime (for seeding random function)
